## Features

- Generate QR codes instantly
- Multiple size options (Small, Medium, Large, XL)
- 6 color presets
- Quick action templates for URLs, emails, phone numbers, and WiFi
- Download QR codes as PNG
- History of last 12 generated codes

## Usage

1. Open `index.html` in your browser
2. Enter text, URL, or data
3. Select size and color
4. Click "GENERATE QR CODE"
5. Download or share your QR code

## Requirements

- Modern web browser
- Internet connection (uses QR Server API)
